# Hướng dẫn sử dụng #

> Project
Sau khi clone project từ github về, giảng viên extract file, tạo mới 01 file .env mới bằng cách tạo file .env với nội dung y chang file .env.example và Save As dưới dạng All File(*) để có thể đặt tên file là .env

> Database
Chạy wampserver và tạo database có tên là stu_henho, sau đó import file database của project vào.

> Lệnh thực thi project Laravel
Truy cập folder của project
Shift + Chuột phải  -> Open Powershell window here
Nhập các lệnh dưới đây vào trong Powershell 
 composer install 
 php artisan key:generate
 php artisan serve
Copy đường dẫn vào URL trình duyệt web.
